# ping-web
pingdom-alternative


**Start the app**
  `node index.js`
  
**Environment changes**
  --Windows(Powershell)
  
  1. **DEBUG**
            `$env:NODE_DEBUG='workers';node index.js`
        
  2. **Production**
            `$env:NODE_ENV="production";node index.js`
        
  3. **Development**
            `$env:NODE_ENV="development";node index.js`
